For each of the following interactions:

\begin{itemize}
\item State whether the interaction is \textbf{valid} or \textbf{invalid}.
\item For \textbf{exotic particles}, the \textbf{quark composition is provided} so you can determine whether the particle is a \textbf{meson or baryon} (and hence its \textbf{baryon number}) and work out its \textbf{charge}.
\item If invalid, identify which conservation law is violated:
\begin{itemize}
\item Charge
\item Lepton number
\item Baryon number
\end{itemize}
\end{itemize}

Only these three conservation laws should be considered.
